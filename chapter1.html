Chapter 1

Functions and Limits

Calculus is the mathematics of change. Before we can talk about how things change, we need a precise language for the things that do the changing — functions — and a tool for examining their behavior up close: the limit.

This chapter has two halves. The first reviews functions: how to read them, where they're defined, and how to build new ones from old. The second introduces the limit, the single idea that everything else in Calc 1 is built on. Every section ends with worked examples whose steps you can reveal one at a time — try the problem yourself first, then peek.

1.1   Functions: the basics

Definition

A function $f$ from a set $D$ (the domain) to a set $Y$ assigns to each input $x \in D$ exactly one output, written $f(x)$.

The single most important word in that definition is exactly one. A rule that gives two outputs for the same input is not a function. The graph test for this is the vertical line test: a curve in the $xy$-plane is the graph of a function of $x$ if and only if no vertical line crosses it more than once.

Three useful ways to describe a function: by a formula (f(x) = x²), by a graph, or by a table. They're all describing the same kind of object — an input/output rule.

Example 1 Evaluating a function
Problem If $f(x) = 2x^2 - 3x + 1$, compute $f(2)$, $f(-1)$, and $f(a+1)$.

Try it yourself first — substitution only.

1

Substitute $x = 2$ into the formula and simplify.

$f(2) = 2(2)^2 - 3(2) + 1 = 8 - 6 + 1 = 3.$
2

Now $x = -1$. Be careful with signs.

$f(-1) = 2(-1)^2 - 3(-1) + 1 = 2 + 3 + 1 = 6.$
3

For $f(a+1)$, treat $a+1$ as a single block and expand.

$f(a+1) = 2(a+1)^2 - 3(a+1) + 1.$

Expand $(a+1)^2 = a^2 + 2a + 1$ and distribute:

$= 2(a^2 + 2a + 1) - 3a - 3 + 1 = 2a^2 + 4a + 2 - 3a - 2 = 2a^2 + a.$
$f(2) = 3,\quad f(-1) = 6,\quad f(a+1) = 2a^2 + a.$

1.2   Domain and range

The domain is the set of all inputs $x$ for which $f(x)$ produces a real number. The range is the set of values $f(x)$ actually takes on. For algebraic functions, the domain is "everything except what breaks the formula." Two things break formulas in Calc 1:

  • division by zero,
  • taking an even root (or a logarithm) of a negative number.
Example 2 Domain with a square root and a fraction
Problem Find the domain of $\displaystyle f(x) = \frac{\sqrt{x-3}}{x-5}.$
1

Identify what could break the formula. There are two issues: the square root in the numerator, and the denominator.

2

The square root requires its argument to be non-negative:

$x - 3 \ge 0 \quad\Longrightarrow\quad x \ge 3.$
3

The denominator must not be zero:

$x - 5 \ne 0 \quad\Longrightarrow\quad x \ne 5.$
4

Intersect both conditions: $x \ge 3$ and $x \ne 5$.

Domain: $[3, 5) \cup (5, \infty)$.
Example 3 Domain of a logarithm
Problem Find the domain of $g(x) = \ln(4 - x^2)$.
1

The logarithm requires a strictly positive argument:

$4 - x^2 > 0.$
2

Solve the inequality. Rearrange:

$x^2 < 4 \quad\Longrightarrow\quad |x| < 2 \quad\Longrightarrow\quad -2 < x < 2.$
Domain: $(-2, 2)$.
Common mistake

For $\ln$, the inequality is strict ($> 0$) because $\ln(0)$ is undefined. For an even root like $\sqrt{\,\cdot\,}$, equality is allowed ($\ge 0$) because $\sqrt{0} = 0$ is fine.

1.3   Combining and composing functions

From two functions $f$ and $g$ we can form $f+g$, $f-g$, $fg$, and $f/g$. The most important new operation, though, is composition: feed the output of one function into the other.

Definition

The composition $f \circ g$ is the function defined by $(f \circ g)(x) = f(g(x))$. Its domain is the set of all $x$ in the domain of $g$ such that $g(x)$ lies in the domain of $f$.

Example 4 Composing two functions
Problem Let $f(x) = x^2 + 1$ and $g(x) = 2x - 3$. Find $(f \circ g)(x)$ and $(g \circ f)(x)$.
1

$(f \circ g)(x)$ means "apply $g$ first, then $f$." So plug $g(x)$ into $f$:

$(f \circ g)(x) = f(g(x)) = f(2x - 3) = (2x-3)^2 + 1.$
2

Expand $(2x-3)^2 = 4x^2 - 12x + 9$:

$(f \circ g)(x) = 4x^2 - 12x + 9 + 1 = 4x^2 - 12x + 10.$
3

For $(g \circ f)(x)$, apply $f$ first then $g$:

$(g \circ f)(x) = g(f(x)) = g(x^2 + 1) = 2(x^2 + 1) - 3 = 2x^2 - 1.$
$(f \circ g)(x) = 4x^2 - 12x + 10$, $\quad(g \circ f)(x) = 2x^2 - 1$. Order matters!

1.4   The intuitive idea of a limit

The central question of a limit is:

The limit question

As $x$ gets close to some number $a$, what number (if any) is $f(x)$ getting close to?

The notation $\displaystyle\lim_{x \to a} f(x) = L$ says: as $x$ approaches $a$ (from either side), $f(x)$ approaches $L$. Crucially, we do not care about $f(a)$ itself — only what happens near $a$.

Example 5 A limit you can guess from a table
Problem Estimate $\displaystyle\lim_{x \to 2} \frac{x^2 - 4}{x - 2}$ by plugging in values close to $2$.
1

First try $x = 2$ directly. The numerator is $0$ and so is the denominator — the expression is undefined. So the function isn't defined at $2$, but the limit may still exist.

2

Try values near $2$ on both sides:

$\begin{array}{c|c} x & f(x) \\ \hline 1.9 & 3.9 \\ 1.99 & 3.99 \\ 1.999 & 3.999 \\ 2.001 & 4.001 \\ 2.01 & 4.01 \\ 2.1 & 4.1 \end{array}$
3

The values are zeroing in on $4$ from both sides.

$\displaystyle\lim_{x \to 2} \frac{x^2 - 4}{x - 2} = 4.$

Tables and graphs are useful for intuition, but they're slow and they can lie (try $\sin(\pi/x)$ near $0$). What we want is an algebraic way to compute limits.

1.5   Computing limits algebraically

Limit Laws

If $\lim_{x\to a} f(x) = L$ and $\lim_{x\to a} g(x) = M$, then

  • $\lim(f \pm g) = L \pm M$
  • $\lim(f \cdot g) = L \cdot M$
  • $\lim(f/g) = L/M$ provided $M \ne 0$
  • $\lim c \cdot f = c \cdot L$ for any constant $c$

For polynomials and rational functions where the denominator is non-zero at $a$, the limit equals $f(a)$. Just plug in.

When direct substitution gives $\dfrac{0}{0}$ — called an indeterminate form — the limit may still exist. You need to do algebra to remove the source of the $0/0$. Two go-to techniques:

  1. Factor and cancel when both numerator and denominator are polynomials sharing a root.
  2. Rationalize by multiplying by the conjugate when a square root is in the way.
Example 6 Plug-in works
Problem Evaluate $\displaystyle\lim_{x \to 2} (3x^2 + 5x - 1)$.
1

This is a polynomial, so direct substitution gives the limit:

$3(2)^2 + 5(2) - 1 = 12 + 10 - 1 = 21.$
$21$.
Example 7 Factor and cancel (0/0)
Problem Evaluate $\displaystyle\lim_{x \to 3} \frac{x^2 - 9}{x^2 - x - 6}.$
1

Try $x = 3$. Numerator: $9 - 9 = 0$. Denominator: $9 - 3 - 6 = 0$. So we have $\frac{0}{0}$ — indeterminate. Time to factor.

2

Factor both:

$\frac{x^2 - 9}{x^2 - x - 6} = \frac{(x-3)(x+3)}{(x-3)(x+2)}.$
3

Both share the factor $(x-3)$, the source of the $0/0$. Cancel it (valid because $x \ne 3$ in the limit):

$= \frac{x+3}{x+2}.$
4

Now substitute $x = 3$:

$\frac{3+3}{3+2} = \frac{6}{5}.$
$\displaystyle\lim_{x \to 3} \frac{x^2 - 9}{x^2 - x - 6} = \frac{6}{5}.$
Example 8 Rationalize the numerator
Problem Evaluate $\displaystyle\lim_{x \to 0} \frac{\sqrt{x+1} - 1}{x}.$
1

Direct substitution gives $\frac{\sqrt{1}-1}{0} = \frac{0}{0}$. We need to clear the square root.

2

Multiply top and bottom by the conjugate $\sqrt{x+1} + 1$:

$\frac{\sqrt{x+1}-1}{x} \cdot \frac{\sqrt{x+1}+1}{\sqrt{x+1}+1} = \frac{(x+1) - 1}{x(\sqrt{x+1}+1)}.$
3

Simplify the numerator: $(x+1) - 1 = x$. Cancel the $x$:

$= \frac{x}{x(\sqrt{x+1}+1)} = \frac{1}{\sqrt{x+1}+1}.$
4

Now substitute $x = 0$ safely:

$\frac{1}{\sqrt{1}+1} = \frac{1}{2}.$
$\displaystyle\lim_{x \to 0} \frac{\sqrt{x+1} - 1}{x} = \frac{1}{2}.$
Example 9 Difference of cubes
Problem Evaluate $\displaystyle\lim_{x \to 1} \frac{x^3 - 1}{x - 1}.$
1

Substitution gives $0/0$. Recognize the difference of cubes:

$x^3 - 1 = (x-1)(x^2 + x + 1).$
2

Cancel $(x-1)$:

$\frac{x^3-1}{x-1} = x^2 + x + 1.$
3

Substitute $x = 1$: $1 + 1 + 1 = 3$.

$3$.
Example 10 Combining fractions
Problem Evaluate $\displaystyle\lim_{h \to 0} \frac{\frac{1}{2+h} - \frac{1}{2}}{h}.$
1

Combine the two fractions in the numerator over the common denominator $2(2+h)$:

$\frac{1}{2+h} - \frac{1}{2} = \frac{2 - (2+h)}{2(2+h)} = \frac{-h}{2(2+h)}.$
2

The whole expression becomes

$\frac{1}{h} \cdot \frac{-h}{2(2+h)} = \frac{-1}{2(2+h)}.$
3

Now $h \to 0$ is safe:

$\frac{-1}{2 \cdot 2} = -\frac{1}{4}.$
$-\dfrac{1}{4}$. (You'll see this exact pattern again — it's the derivative of $1/x$ at $x=2$.)

1.6   One-sided limits

Sometimes a function behaves differently as you approach $a$ from the left ($x \to a^-$) than from the right ($x \to a^+$). We write

$\lim_{x \to a^-} f(x) \quad\text{and}\quad \lim_{x \to a^+} f(x).$
Two-sided existence

$\displaystyle\lim_{x \to a} f(x) = L$ if and only if both one-sided limits exist and equal $L$.

Example 11 Absolute value — the classic two-sided failure
Problem Investigate $\displaystyle\lim_{x \to 0} \frac{|x|}{x}.$
1

Recall $|x| = x$ when $x > 0$ and $|x| = -x$ when $x < 0$.

2

Right-hand limit ($x > 0$):

$\lim_{x \to 0^+} \frac{|x|}{x} = \lim_{x \to 0^+} \frac{x}{x} = 1.$
3

Left-hand limit ($x < 0$):

$\lim_{x \to 0^-} \frac{|x|}{x} = \lim_{x \to 0^-} \frac{-x}{x} = -1.$
4

The two sides disagree, so the two-sided limit does not exist.

$\displaystyle\lim_{x \to 0^+} \frac{|x|}{x} = 1$, $\;\displaystyle\lim_{x \to 0^-} \frac{|x|}{x} = -1$, two-sided limit does not exist.
Example 12 Piecewise function
Problem Let $f(x) = \begin{cases} x^2 + 1, & x < 1 \\ 3x - 1, & x \ge 1 \end{cases}$. Does $\displaystyle\lim_{x \to 1} f(x)$ exist?
1

From the left, $x < 1$ so we use $x^2 + 1$:

$\lim_{x \to 1^-} f(x) = 1^2 + 1 = 2.$
2

From the right, $x \ge 1$ so we use $3x - 1$:

$\lim_{x \to 1^+} f(x) = 3(1) - 1 = 2.$
3

Both one-sided limits equal $2$, so the two-sided limit exists.

$\displaystyle\lim_{x \to 1} f(x) = 2.$

1.7   Limits involving infinity

"Infinity" shows up in two different ways in limits:

  • Infinite limits: $\lim_{x\to a} f(x) = \pm\infty$ — the function grows without bound as $x$ approaches a finite point $a$. This is the picture of a vertical asymptote.
  • Limits at infinity: $\lim_{x\to\pm\infty} f(x) = L$ — we ask what $f$ does for very large (or very negative) $x$. This is the picture of a horizontal asymptote.
Example 13 A vertical asymptote
Problem Investigate $\displaystyle\lim_{x \to 2} \frac{1}{(x-2)^2}.$
1

As $x \to 2$, the denominator $(x-2)^2$ approaches $0$. The squared term is always positive, so the denominator is a tiny positive number.

2

$\dfrac{1}{\text{tiny positive}}$ is huge and positive. So from either side, the value grows without bound:

$\lim_{x \to 2} \frac{1}{(x-2)^2} = +\infty.$
$+\infty$. The function has a vertical asymptote at $x = 2$.
Example 14 Horizontal asymptote of a rational function
Problem Evaluate $\displaystyle\lim_{x \to \infty} \frac{3x^2 + 2x}{5x^2 - x + 1}.$
1

For large $x$, the highest-power terms dominate. Divide every term by $x^2$ (the highest power in the denominator):

$\frac{3x^2 + 2x}{5x^2 - x + 1} = \frac{3 + \frac{2}{x}}{5 - \frac{1}{x} + \frac{1}{x^2}}.$
2

As $x \to \infty$, every term with $x$ in the denominator vanishes:

$\to \frac{3 + 0}{5 - 0 + 0} = \frac{3}{5}.$
3

Quick shortcut: for a rational function, when the numerator and denominator have the same degree, the limit at $\pm\infty$ is the ratio of leading coefficients.

$\dfrac{3}{5}$. The line $y = 3/5$ is a horizontal asymptote.
Example 15 A tricky one with a square root
Problem Evaluate $\displaystyle\lim_{x \to \infty} \left(\sqrt{x^2 + 1} - x\right).$
1

Both terms blow up, giving an indeterminate $\infty - \infty$. Rationalize by multiplying by the conjugate over itself:

$\left(\sqrt{x^2+1} - x\right) \cdot \frac{\sqrt{x^2+1} + x}{\sqrt{x^2+1} + x} = \frac{(x^2+1) - x^2}{\sqrt{x^2+1} + x}.$
2

The numerator simplifies to $1$:

$= \frac{1}{\sqrt{x^2+1} + x}.$
3

As $x \to \infty$, the denominator $\to \infty$, so the whole fraction $\to 0$.

$0$.

1.8   Continuity

Definition

$f$ is continuous at $a$ if all three of the following hold:

  1. $f(a)$ is defined,
  2. $\displaystyle\lim_{x \to a} f(x)$ exists,
  3. $\displaystyle\lim_{x \to a} f(x) = f(a)$.

Informally: you can draw the graph through $x = a$ without lifting your pencil. When continuity fails, the trouble comes in flavors:

  • Removable — a single missing or "wrong" point. The limit exists, but doesn't match $f(a)$. Fixable by redefining one value.
  • Jump — the two one-sided limits exist but disagree.
  • Infinite — the function blows up to $\pm\infty$.
Example 16 Classify the discontinuity
Problem Where is $\displaystyle f(x) = \frac{x^2 - 1}{x - 1}$ discontinuous? What kind?
1

$f$ is undefined at $x = 1$ (denominator $0$), so something fails there. Test the limit.

2

Factor: $\dfrac{x^2-1}{x-1} = \dfrac{(x-1)(x+1)}{x-1} = x + 1$ (for $x \ne 1$).

3

So $\displaystyle\lim_{x\to 1} f(x) = 2$, which is a finite number — but $f(1)$ doesn't exist. The limit exists, but $f$ doesn't match it at $x=1$. That's a removable discontinuity.

Removable discontinuity at $x = 1$. Defining $f(1) = 2$ would make $f$ continuous there.
Example 17 Make a piecewise function continuous
Problem For what value of $k$ is $f(x) = \begin{cases} x^2 + k, & x \le 2 \\ 3x - 1, & x > 2 \end{cases}$ continuous everywhere?
1

Each piece is a polynomial, so continuous on its own. The only worry is the joining point $x = 2$.

2

For continuity at $2$, the left value, right limit, and $f(2)$ must all agree:

$\lim_{x \to 2^-} f(x) = 4 + k, \qquad \lim_{x \to 2^+} f(x) = 5.$
3

Set them equal:

$4 + k = 5 \quad\Longrightarrow\quad k = 1.$
$k = 1$.

1.9   The Squeeze Theorem and trig limits

Squeeze (Sandwich) Theorem

If $g(x) \le f(x) \le h(x)$ for all $x$ near $a$ (except possibly at $a$), and $\displaystyle\lim_{x \to a} g(x) = \lim_{x \to a} h(x) = L$, then $\displaystyle\lim_{x \to a} f(x) = L$.

The Squeeze Theorem is the right tool when a function wiggles too wildly to plug in or factor — typically because it has $\sin$ or $\cos$ of something like $1/x$ in it.

Example 18 A wild oscillation tamed
Problem Show that $\displaystyle\lim_{x \to 0} x^2 \sin\!\left(\tfrac{1}{x}\right) = 0.$
1

$\sin(1/x)$ oscillates between $-1$ and $1$ as $x \to 0$, so we cannot just "plug in" the inside. But we can bound the whole thing.

2

For any $x \ne 0$, $-1 \le \sin(1/x) \le 1$. Multiply by $x^2$, which is non-negative (so the inequalities don't flip):

$-x^2 \le x^2 \sin(1/x) \le x^2.$
3

Both outer functions have limit $0$ as $x \to 0$. By the Squeeze Theorem, so does the middle.

$\displaystyle\lim_{x \to 0} x^2 \sin(1/x) = 0.$

Two trig limits you must memorize

Fundamental trig limits
$\lim_{x \to 0} \frac{\sin x}{x} = 1, \qquad \lim_{x \to 0} \frac{1 - \cos x}{x} = 0.$

These are not optional. Almost every trig limit you'll meet in Calc 1 reduces to one of these two.

Example 19 Match the pattern of sin x / x
Problem Evaluate $\displaystyle\lim_{x \to 0} \frac{\sin(5x)}{3x}.$
1

We want the inside of the sine to match the denominator. The trick is to multiply and divide by $5$:

$\frac{\sin(5x)}{3x} = \frac{5}{3} \cdot \frac{\sin(5x)}{5x}.$
2

As $x \to 0$, the new "inside" $u = 5x$ also goes to $0$, so $\dfrac{\sin(5x)}{5x} \to 1$.

3

Therefore the limit is $\dfrac{5}{3} \cdot 1 = \dfrac{5}{3}.$

$\dfrac{5}{3}.$
Example 20 Combine the two trig limits
Problem Evaluate $\displaystyle\lim_{x \to 0} \frac{1 - \cos x}{x^2}.$
1

Multiply top and bottom by the conjugate $1 + \cos x$:

$\frac{1 - \cos x}{x^2} \cdot \frac{1 + \cos x}{1 + \cos x} = \frac{1 - \cos^2 x}{x^2(1 + \cos x)}.$
2

Use $1 - \cos^2 x = \sin^2 x$:

$= \frac{\sin^2 x}{x^2(1 + \cos x)} = \left(\frac{\sin x}{x}\right)^2 \cdot \frac{1}{1 + \cos x}.$
3

Take the limit. The first piece goes to $1^2 = 1$ and the second piece goes to $\dfrac{1}{1 + 1} = \dfrac{1}{2}$:

$\to 1 \cdot \frac{1}{2} = \frac{1}{2}.$
$\dfrac{1}{2}.$ This limit is the secret ingredient behind the second derivative of $\cos x$ at $0$.

Chapter recap

In a sentence: a function is a rule, a limit is a question about behavior near a point, and continuity is the rule that makes the limit and the function agree.

The toolbox for computing limits in Calc 1:

  1. Plug in. If the function is a polynomial, rational (away from zeros), or any algebraic combination of continuous functions, just substitute.
  2. If you get $\tfrac{0}{0}$, do algebra. Factor and cancel, rationalize, combine fractions — whatever kills the shared root.
  3. For $\pm\infty$, compare growth. Divide by the highest power in the denominator; the leading coefficients win.
  4. Wild oscillation? Squeeze it.
  5. Trig at zero? Reshape into $\dfrac{\sin u}{u}$ or $\dfrac{1 - \cos u}{u}$.

In Chapter 2 we use these limits to define the derivative.
